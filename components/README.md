# FlexPass - Event Ticketing Platform 🎟️

FlexPass is a modern ticketing platform built for the Nigerian market, allowing hosts to create events and attendees to buy tickets seamlessly.

## 🚀 Features
- **User Authentication:** Secure Signup/Login for Hosts (Supabase Auth).
- **Event Management:** Hosts can create, manage, and track events.
- **Ticketing System:** Dynamic QR-coded tickets with inventory management.
- **Payments:** Integrated Paystack payment gateway for instant settlements.
- **Dashboard:** Real-time analytics for sales and revenue tracking.
- **Email Notifications:** Automated ticket receipts via Resend API.

## 🛠️ Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Database:** Supabase (PostgreSQL)
- **Payments:** Paystack
- **Emails:** Resend
